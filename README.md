# 8-Puzzle Solver

An automated solver that translates images of physical 8-puzzles into digital states and finds the fastest way to solve them.

## Features
* **Image Processing:** Uses OpenCV to segment the 3x3 grid and extract individual tiles.
* **Digit Recognition:** Features a TensorFlow model to identify digits (1 to 8) on the tiles.
* **Pathfinding:** Implements search algorithms to calculate the optimal path to the goal state.

## Tech Stack
* Python
* OpenCV
* TensorFlow
* NumPy
